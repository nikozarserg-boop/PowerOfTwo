# PowerOfTwo

Проект для решения различных задач и сравнения их реализации на разных языках программирования. Цель проекта - продемонстрировать особенности синтаксиса, производительности и подходов к решению одинаковых задач на различных языках.

## языки

- **Python**
- **C**
- **C++**
- **C#**
- **Java**
- **Fortran**
- **Assembly**
- **Bash/PowerShell**
- **Objective-C**
- **Pascal**
- **Dart**
- **F#**
- **BASIC**
- **Visual Basic**
- **D**
